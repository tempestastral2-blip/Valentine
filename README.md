# Valentines (Simple Web Page)

This is a small static web page that shows a Valentine's envelope, a short interactive letter, and three gift boxes.

How to open locally

- Double-click `index.html` or open it in Chrome/Edge.
- Or serve locally from the folder (recommended):

  ```powershell
  # from the project folder
  python -m http.server 8000
  # then open http://localhost:8000
  ```

What this repo contains

- `index.html` — page markup
- `style.css` — styles
- `script.js` — interactivity
- image assets (envelope.png, cat_heart.gif, cat_dance.gif, yes.png, no.png, heart-bg.jpg, window.png)

Share options

- Create a ZIP of the folder and send it.
- Push to GitHub and enable GitHub Pages.
- Deploy via Netlify or Vercel.

Notes

- Keep the folder structure intact when sharing (assets are referenced with relative paths).
